# 中国高校层次划分与标签体系

985工程：1998年5月提出，共39所高校入选，代表了中国高等教育的最高层次。包括北京大学、清华大学、复旦大学、上海交通大学、浙江大学、南京大学、中国科学技术大学、哈尔滨工业大学、西安交通大学等C9联盟高校以及其余30所重点大学。985高校在经费投入、师资力量、科研平台、保研率、就业认可度等方面具有显著优势。

211工程：面向21世纪重点建设约100所高等学校和一批重点学科，共115所高校（含39所985高校）。211高校遍布全国各省份，是各区域的龙头院校。

双一流：2017年启动的新一轮高等教育重点建设工程，分为一流大学建设高校（42所）和一流学科建设高校（95所）。2022年第二轮名单公布，不再区分一流大学和一流学科建设高校，统一以学科为基础进行建设。

其他重要标签：
- 省部共建：由教育部与省级政府共同建设的高校，通常在本省影响力较大。
- 中西部高校基础能力建设工程（小211）：重点支持100所中西部本科高校。
- 卓越工程师/医生/法律人才/农林人才教育培养计划：在特定领域有特色优势的高校。

选校建议：除了看整体层次标签外，更应关注具体专业的学科评估结果。教育部学位中心第四轮学科评估（2017年公布）和第五轮学科评估（2022年起陆续公布）将学科分为A+至C-共9档，这是判断学校某专业实力的重要依据。

对于分数在边缘线的考生，可以关注"双一流学科建设高校"的特色优势专业，这些专业的实力往往超过部分985高校的同类专业。
