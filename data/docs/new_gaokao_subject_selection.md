# 新高考选科与志愿填报关系

新高考改革后，选科组合直接影响可报考的大学专业范围。考生在高一或高二选科时，就需要了解目标专业的选科要求。

3+3模式（浙江、上海、北京、天津、山东、海南）：语数外3门必考 + 从物化生政史地中任选3门。专业选科要求灵活多样，同一专业不同学校的选科要求可能不同。

3+1+2模式（河北、辽宁、江苏、福建、湖北、湖南、广东、重庆等）：语数外3门 + 物理或历史首选1门 + 从化生政地中再选2门。物理类考生可报考绝大多数理工农医类专业，历史类考生以文史哲法经管类专业为主。

专业选科要求的常见组合：
- 要求必选物理：计算机类、电子信息类、机械类、土木类、电气类等绝大多数工科专业。
- 要求物理+化学：临床医学、化学工程、材料科学、药学等。
- 要求必选历史：部分高校的考古学、中国史、世界史等。
- 不限选科：大部分文学、哲学、法学、经济学、管理学、教育学等专业。

填报志愿时，首选需要确认目标专业的选科要求是否与自己匹配。教育部《普通高校本科招生专业选考科目要求指引》是权威参考文件。不同学校同一专业的选科要求可能不同，务必逐一核实。

选科组合覆盖率参考（3+1+2模式）：物理+化学+生物组合的专业覆盖率最高，可达95%以上；物理+化学+X组合覆盖率约90%-95%；历史+政治+地理组合专业覆盖率约50%-60%，主要集中在文史哲法经管类。
