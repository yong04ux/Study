# 各省高考总分与录取规则差异

全国各省高考总分和计分方式不完全相同，填报志愿时必须区分。主要差异如下：

总分差异：
- 750分制（大多数省份）：语数外各150分 + 选考科目各100分（3+1+2模式）或3门各100分（3+3模式）。包括广东、江苏、河北、湖北、湖南、福建、山东、浙江等。
- 660分制：上海。语数外各150分（450分）+ 3门等级考各70分（210分）。
- 900分制：海南。标准分制度，各科原始分转换为标准分后相加。

志愿填报方式差异：
- 院校专业组模式（多数新高考省份）：一所院校设置若干专业组，每组内专业选科要求相同，考生按专业组填报。如广东本科批可填45个院校专业组志愿。
- 专业+院校模式（浙江、山东、河北、辽宁、重庆等）：以具体专业为志愿单位，直接填报某校某专业，浙江可填80个志愿，山东可填96个志愿。
- 传统院校+专业模式（少数未改革省份）：先选学校，再选该校的若干专业。

录取规则差异：
- 各省本科线/一本线的划定方式不同，与考生人数和计划数比例有关。
- 部分省份有"分段录取"机制，如浙江按实考人数的20%、60%、90%分段。
- 农村专项、地方专项等特殊类型的实施区域和报考条件各省不同。

重要提醒：在参考往年的录取分数线和位次时，务必注意只参考同一省份、同一科类的数据。不同省份之间的分数和位次不可直接比较。
