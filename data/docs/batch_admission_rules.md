# 高考录取批次与投档规则详解

中国高考录取分为多个批次，不同省份设置略有差异，但总体框架逐年趋同。主要批次包括：

提前批：包括军队院校、公安院校、司法院校、飞行学员、公费师范生、优师专项、农村订单定向医学生、部分艺术体育类专业等。提前批未被录取不影响后续批次投档，是增加录取机会的重要批次。

本科批：新高考省份已将原本科一批和本科二批合并为本科批。少数省份仍保留本科一批、本科二批。本科批是绝大多数考生录取的主要批次。投档方式以平行志愿为主，遵循"分数优先、遵循志愿、一次投档"原则。

专科批：包括高职高专院校及本科院校的专科专业。一般在本科录取全部结束后进行。

投档规则核心概念：
- 平行志愿：按考生分数从高到低排序，依次检索每个考生的志愿。当检索到某一志愿尚有计划余额时即投档，本批次后续志愿不再检索。
- 顺序志愿：按志愿顺序分批投档，先投第一志愿，第一志愿录取结束后再投第二志愿。目前仅在提前批部分类型中使用。
- 一次投档：平行志愿模式下，考生档案只能投出一次。如果被投档后因不服从调剂、身体条件不符等原因被退档，只能参加征集志愿或下一批次录取。

退档的常见原因：不服从专业调剂、单科成绩不达标、身体条件不符合专业要求、外语语种不匹配、政审面试不合格等。填报时务必仔细核对目标院校招生章程。
